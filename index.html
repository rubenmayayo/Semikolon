What is Semikolon?

Semikolon is a responsive fixed grid system that allows to divide your layout in rows, and each row in up to 12 columns. Using the apropiate class, columns can be combined and shifted to create diferent containers.

Usage

Simply add semikolon.min.css to your page:

<link rel="stylesheet" href="css/semikolon.min.css">

This is the code used to generate the previous columns and rows:
